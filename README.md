# Minimalist-Homepage-BE

浏览器扩展版选择

例如 1x.2x.3x 其中1x对应极简首页版本,3x:正式版为3 综合预览版为2 预览版为1

## 功能特点

🎨 深色/浅色主题切换 - 支持自动保存主题偏好

🔍 多搜索引擎支持 - 必应搜索和百度搜索一键切换

📝 搜索历史记录 - 自动保存最近5条搜索记录

📱 响应式设计 - 完美适配桌面和移动设备

🎯 简洁界面 - 专注于搜索功能，无多余干扰

## 使用方法
直接在输入框中输入搜索内容，按回车或点击"搜索"按钮

点击搜索选项切换搜索引擎（默认使用必应搜索）

点击右上角月亮/太阳图标切换深色/浅色模式

查看和使用下方的搜索历史记录快速搜索
